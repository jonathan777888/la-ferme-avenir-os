# La Ferme de l’Avenir OS

Studio IA autonome pour produire, organiser et contrôler l’univers créatif de La Ferme de l’Avenir.

## Objectif

Construire une plateforme capable de gérer :

- la bible créative ;
- les personnages ;
- les lieux ;
- les règles du canon ;
- les scènes ;
- les prompts manga, image et animation ;
- la cohérence narrative et artistique.

## Modules prévus

- Frontend React
- Backend Node.js / Express
- Service IA Flask
- Base de données PostgreSQL
- Docker
- CI/CD
